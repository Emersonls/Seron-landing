# Security Policy

## Project Security Commitment

We take the security of this project seriously. Protecting user data, platform integrity, and system stability is a top priority.

## Supported Versions

Only the latest production version is actively maintained and supported with security updates.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly and privately.

Do NOT open a public issue.

Send your report including:

- Detailed description of the vulnerability
- Steps to reproduce
- Screenshots or proof of concept (if applicable)
- Potential impact assessment

Contact Email: emersonlsiva02@email.com

We aim to respond within 48 hours and will keep you informed throughout the resolution process.

## Responsible Disclosure

We appreciate responsible disclosure and will acknowledge verified security reports. Our goal is to resolve critical issues as quickly as possible to maintain trust and reliability.

Thank you for helping strengthen this project.
